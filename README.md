# Birthday Site

A beautiful React birthday site built with Vite.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:5173`

## Project Structure

- `src/App.jsx` - Main birthday site component
- `public/` - Static assets (images, videos, music)
  - `img/` - Images
  - `vid/` - Videos
  - `music/` - Audio files

## Adding Media Assets

Place your media files in the corresponding public folders:
- Images: `public/img/`
- Videos: `public/vid/`
- Music: `public/music/`

Update the imports in `src/App.jsx` if needed.

## Build

To build for production:
```bash
npm run build
```

## Features

- Interactive birthday letter
- Photo and video galleries
- Countdown timer
- Custom animations and effects
- Responsive design
